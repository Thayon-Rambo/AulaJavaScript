Atalhos do VS Code

Atalhos gerais

ctrl + espaço: chama o autocompletar de um contexto

Vamos ver alguns atalhos legais do VS Code que vão deixar a nossa vida mais alegre, supimpa e feliz

alt + z: Quebra automatica de linha

alt + setas cima/baixo: Move a seleção para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a liha em que o cursor.

shift + alt + setas cima/baixo: Copia a seleção para cima/baixo no arquivo. Se nada estiver selecionado, copia a liha em que o cursor.

ctrl + ; : Comentario/descomentario
